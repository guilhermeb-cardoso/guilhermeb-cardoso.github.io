Personal Projects

Jak Mars

Jak Mars was my solo final project of the Unreal Engine Programming Course where I focused on building a complete third-person action adventure in Unreal Engine 5. The main challenge was creating scalable gameplay systems using Blueprint architecture, while also delivering responsive combat, AI, and a functional user interface.

  • Implemented AI with Behavior Trees & Blackboards for dynamic decision-making.
  • Developed modular gameplay mechanics and interactive objects using a Blueprint-driven approach, ensuring smooth character movement, responsive abilities, and engaging player-world interactivity.
  • Integrated UMG UI: HUDs, menus, health bars, and overlays.
  • Built environments with Nanite meshes, Lumen lighting and foliage.
  • Designed adaptive audio using Unreal Audio Engine with spatialized SFX and event-driven music.

Golden Taco

Golden Taco was created during the Global Game Jam in a 48-hour sprint. The project emphasized rapid prototyping under strict time constraints, requiring quick technical decisions and effective implementation. My primary responsibilities were gameplay programming and input systems, alongside integrating UI/UX elements to deliver a complete vertical slice.

  • Implemented player controls using the Enhanced Input System for modular bindings and adaptability.
  • Developed responsive UMG menus and in-game HUDs within the jam timeframe.
  • Handled GameMode and PlayerController logic for seamless state transitions (pause, restart, game-over).

  • Modeled basic assets in Blender for early-level prototyping.
  • Explored game rules, dynamics, genres, platforms, and player types.
  • Compared Unity and Unreal from a designer’s perspective.
  • Analyzed game industry history and aesthetics to contextualize modern design.
